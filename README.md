# cts-DoNet

This repository contains directories and resources for the .NET upskilling and deepskilling learning paths.

## Repository Structure

The repository is divided into two primary sections:

### 1. Upskilling (`/Upskilling`)
Resources and tasks associated with the fundamental upskilling path.

### 2. Deepskilling (`/Deepskilling`)
Detailed topics and modules covering advanced concepts and practices:
- **Advanced SQL server**: Advanced database scripting, optimization, and administration concepts.
- **Angular**: Modern frontend web development using the Angular framework.
- **Engineering concepts**: General software engineering design principles, design patterns, and practices.
- **Entity Framework Core**: Object-Relational Mapping (ORM) for .NET.
- **GIT**: Version control guidelines and practices.
- **Microservices**: Microservice architecture patterns, communication protocols, and design.
- **NUnit and Moq**: Unit testing and mocking practices in .NET.
- **React**: Modern frontend web development using React.
- **WebApi**: Building RESTful APIs with ASP.NET Core Web API.
